# pages-www-leisu

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面各自独立，不针对任何特定域名或网站，旨在以静态方式存储和展示分散的 Web 内容。

所有页面均为纯前端 HTML 文件，不依赖后端服务，可直接在浏览器中打开或通过静态托管服务访问。

## 目录说明

```
pages-www-leisu/
├── src/              # 页面源代码（HTML、CSS、JS 等）
├── docs/             # 构建或归档后的页面副本（可选）
├── assets/           # 共享资源（图片、字体等）
├── README.md         # 本文件
└── .gitignore        # Git 忽略规则
```

- `src/`：存放所有页面的原始文件，按需组织子目录。
- `docs/`：如果启用 GitHub Pages，可将构建后的页面放置于此。
- `assets/`：页面之间的共用静态资源。

## 页面归档说明

本仓库中的每个 HTML 页面均为独立归档单元，内容互不关联。页面可能包含特定的样式、脚本或嵌入资源，但所有依赖均已本地化或通过 CDN 引用。

归档目的：
- 保存某类页面的原始结构
- 便于离线查看或分发
- 作为 Web 页面片段的历史记录

注意：归档页面不保证与任何在线版本一致，也不代表任何外部实体的立场。

## 使用方式

1. 克隆仓库：
   ```bash
   git clone https://github.com/your-username/pages-www-leisu.git
   ```
2. 在浏览器中直接打开 `src/` 下的任意 `.html` 文件即可查看。
3. 若配置了 GitHub Pages，可通过仓库设置中的 Pages 功能将 `docs/` 目录发布为静态站点。

## 维护说明

- 新增页面请放入 `src/` 对应子目录，并保持文件结构清晰。
- 请勿提交与页面无关的大型二进制文件。
- 如有资源更新，请同步修改引用路径。
- 本仓库不接受外部贡献，仅供内部归档使用。
